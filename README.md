## ⚖️ Court-MOE

**Mixture-of-Experts based Legal Judgment Prediction System**

Court-MOE is an AI system designed to analyze **Indian legal judgments and predict case outcomes** using transformer-based NLP and **Mixture-of-Experts architectures**.

Key Highlights:

* Implemented **LegalBERT-based document encoding** for legal text representation
* Designed a **Mixture-of-Experts model with court-specific expert networks**
* Built a **router network to dynamically select the most relevant expert model**
* Developed preprocessing pipelines for **long-form legal documents**
* Applied training techniques including **mixed precision, SWA, and EMA**

Tech Stack:
`Python` `PyTorch` `HuggingFace Transformers` `LegalBERT` `MoE` `Pandas` `NumPy`
