# тЪЦя╕П Court-MOE

**Mixture-of-Experts based Legal Judgment Prediction System**

Court-MOE is an AI system designed to analyze **Indian court judgments and predict legal outcomes** using transformer-based NLP and a **Mixture-of-Experts architecture**.

The system processes long-form legal documents and dynamically routes them to **court-specialized expert models** trained on different judicial levels.

### Key Features

* Implemented **LegalBERT-based document encoding** for legal text representation
* Designed a **Mixture-of-Experts architecture with five court-specific expert networks**
* Built a **router network for dynamic expert selection**
* Developed **custom legal tokenization pipelines for long legal documents**
* Applied **advanced training techniques** including mixed precision, SWA, and EMA
* Trained and evaluated models on **large-scale Indian legal judgment datasets**

### Tech Stack
